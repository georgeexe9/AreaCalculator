Валидации
ВАЛИДАЦИИ ЗА ОСНОВНИ 2D ФИГУРИ

Фигура            | Необходими валидации (какво да провериш)
------------------|--------------------------------------------------------------
Правоъгълник      | - Всички страни > 0
                  | - Противоположните страни са равни (ако се изисква)
                  | - Не допускай страни с дължина 0 или отрицателни
Квадрат           | - Всички страни > 0
                  | - Всички страни равни
Триъгълник        | - Всички страни > 0
                  | - Сума от двете по-къси страни > третата страна (неравенството на триъгълника)
                  | - Ако има ъгли — стойности между 0 и 180°
Правоъгълен триъгълник | - Всички страни > 0
                  | - Питагоровото равенство (като проверка)
                  | - Ъглите са 90°, 0° < другите < 90°
Равностранен триъгълник | - Всички страни > 0
                  | - Всички страни равни
Равнобедрен триъгълник | - Всички страни > 0
                  | - Две страни равни
                  | - Неравенството на триъгълника
Трапец            | - Всички страни > 0
                  | - Височината > 0
                  | - Страните, които не са основи, трябва да могат да свържат основите (проверка с проекции)
                  | - Височината не трябва да превишава страните, които са наклонени
                  | - Разликата между основите трябва да е по-малка или равна на сумата от хоризонталните проекции на наклонените страни
Овал (Елипса)     | - Двата радиуса (основни оси) > 0
Кръг              | - Радиус > 0
Паралелограм      | - Всички страни > 0
                  | - Противоположните страни равни
                  | - Ъглите > 0 и < 180°
                  | - Височината > 0 (ако се изисква)
Ромб              | - Всички страни > 0
                  | - Всички страни равни
                  | - Ъглите > 0 и < 180°
